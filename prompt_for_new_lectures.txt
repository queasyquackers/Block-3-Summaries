Task: Using the provided PDF slides and the associated Transcript for Lecture L[#], generate a structured study guide, an assessment, glossary, and a high-yield visual deck.
Constraint: DO NOT use outside information. Rely ONLY on the provided text (transcripts folder) and slides (pdf folder).

Part 1: The Summary (Format for MedSchoolIndex)
Create a summary with a cohesive, connective narrative. Do not just list bullet points; explain the "why" and "how" to link structural and functional relationships.

**HEADER FORMAT:**
Start the summary with:
# Lecture #[Number]: [Title]

**Lecturer:** [Name]
**Session:** Lecture #[Number]

-----

**CRITICAL FORMATTING INSTRUCTIONS:**

1. **Slide Integration:** At the beginning of each major heading or section where specific slides are relevant, you MUST use the following specific syntax.
   
   Syntax:
   :::slides [Slide Numbers]
   * **Slide X-Y:** Description of what is shown
   * **Slide Z:** Another key point
   :::

   Example:
   :::slides 1-5
   * **Slide 1-3:** Axial MRI showing orbital fat; Osteology of orbit walls
   * **Slide 4-5:** Orbital septum; Blowout fractures into ethmoidal/maxillary sinuses
   :::

2. **Clinical Correlates:** Whenever you mention a Clinical Correlate, Drug Toxicity/Adverse Effect, or a specific Disease State, you MUST use the following specific syntax.
   
   Syntax:
   :::clinical
   **Clinical Correlate: [Condition Name]**

   [Paragraph describing the condition, mechanism, and clinical findings.]

   **Clinical Signs:**
   - Sign 1
   - Sign 2
   :::

3. **Headings:**
   - Use `#` for the Main Title.
   - Use `##` for Major Parts (e.g., "Part 1: Introduction").
   - Use `###` for Sections.
   - Use `####` for Sub-sections.

4. **Tables:** Use standard Markdown tables to organize complex comparisons or classifications.

5. **Emphasis:** Use bold (`**text**`) for important terms.
